## 二、选择填空高频概念

### ★★★ 必考点

#### 1. MATLAB名称来源
- MATLAB = **Matrix + Laboratory**（矩阵实验室）
- 最初由Cleve Moler开发，主要用于**数值计算**

#### 2. 变量名命名规则
- 由**一个字母**引导，后面可以为其他字符（字母、数字、下划线）
- **区分大小写**：Abc ≠ ABc
- 有效长度：63个字符
- **关键字不能用作变量名**：for、end、if、while、function、else、break、continue等

#### 3. 特殊矩阵创建函数

| 函数 | 功能 |
|------|------|
| zeros(m,n) | 创建m×n全零矩阵 |
| ones(m,n) | 创建m×n全1矩阵 |
| eye(n) | 创建n阶单位矩阵 |
| rand(m,n) | [0,1]均匀分布随机矩阵 |
| randn(m,n) | 正态分布随机矩阵 |
| magic(n) | 创建n阶魔方矩阵 |
| linspace(a,b,n) | 产生线性分布的n个元素向量 |
| logspace(a,b,n) | 产生对数分布的n个元素向量 |

#### 4. 点运算与矩阵运算的区别（高频！）

| 运算 | 数组运算（点运算） | 矩阵运算 |
|------|-------------------|----------|
| 乘法 | A.*B — 对应元素相乘 | A*B — 线性代数矩阵乘法 |
| 除法 | A./B — 对应元素相除 | A/B — 矩阵右除（AX=B的解） |
| 乘方 | A.^k — 每个元素k次方 | A^k — k个矩阵A相乘 |
| 转置 | A.' — 非共轭转置 | A' — 共轭转置 |

> ⚠️ **易错点：** 加减运算没有点运算，数组和矩阵的加减是一样的，都是对应元素相加减。

#### 5. 矩阵寻址
- **A(i,j)**：第i行第j列的元素
- **A(i,:)**：第i行所有元素
- **A(:,j)**：第j列所有元素
- **A(1:3,:)**：第1至第3行的所有列
- **A(:,1:3)**：所有行的第1至第3列
- **A(end,:)**：最后一行
- **A(:,end)**：最后一列
- **A(:)**：将矩阵按列堆叠成列向量
- **A(1:2:end)**：步长为2的索引

#### 6. M文件类型

| 项目 | M脚本文件 | M函数文件 |
|------|-----------|-----------|
| 输入/输出参数 | 不接收，不返回 | 接收输入，可返回输出 |
| 变量作用域 | 全局变量，保存在基本工作空间 | 默认内部变量为局部变量 |
| 第一行 | 无特殊要求 | 以 `function` 关键字开头 |
| 适用情况 | 需多次执行的一系列命令 | 需多次执行且需要输入/输出参数 |

#### 7. 绘图函数
- **plot(x,y)**：绘制二维曲线
- **hold on**：在同一图中继续画多条线
- **hold off**：关闭保持功能
- **subplot(m,n,p)**：将窗口分成m×n个子图，当前在第p个
- **grid on / grid off**：显示/隐藏网格
- **xlabel('str') / ylabel('str')**：横/纵坐标标签
- **title('str')**：图形标题
- **legend('str1','str2')**：图例
- **axis([xmin xmax ymin ymax])**：设置坐标范围
- **figure**：创建新的图形窗口
- **clf / cla**：清除图形/清除坐标轴

#### 8. 截面惯性矩
- **定义**：I_xx = ∫_A y² dA （对x轴的面积惯性矩）
- **单位**：m⁴ 或 mm⁴
- **矩形对底边x轴**：I_xx = bh³/3
- **矩形对形心x轴**：Ī_xx = bh³/12
- **三角形对底边x轴**：I_xx = bh³/12
- **平行移轴定理**：I_xx = Ī_xx + d_y²·A

> 💡 **记忆口诀：** 对底边是1/3，对形心是1/12。b是宽（平行于轴），h是高（垂直于轴）。高的三次方！

- **无穷多解**：rank(A) = rank(C) < n
- **无解**：rank(A) ≠ rank(C)（矛盾方程，用最小二乘解）
- 左除法：x = A\b

#### 8. 符号计算 vs 数值计算

| 对比项 | 符号计算 | 数值计算 |
|--------|----------|----------|
| 运算对象 | 符号（变量、函数） | 具体数值（整数、浮点数） |
| 计算规则 | 遵循数学推导规则 | 遵循数值运算规则 |
| 结果形式 | 符号表达式 | 具体数值 |
| 精度 | 精确解析解 | 近似数值解 |

## 三、简答题分点答案

### ★★★ 必考点

#### 1. MATLAB的主要功能有哪些？
1. **数值分析与计算**：矩阵运算、线性代数、数值积分等
2. **数据分析和可视化**：数据处理、统计分析、二维/三维绘图
3. **编程和算法开发**：M文件编程、流程控制、函数开发
4. **应用程序开发和部署**：GUI设计、独立应用程序生成

#### 2. 数组运算（点运算）和矩阵运算的异同点？

**相同点：**
- 矩阵是特殊的二维数组，存储形式相同
- 加减运算规则相同，都是对应元素相加减

**不同点：**
- **运算规则不同**：数组运算遵循MATLAB自定义规则（对应元素运算），矩阵运算遵循线性代数规则
- **乘法不同**：数组用.*（对应元素相乘），矩阵用*（线性代数乘法）
- **除法不同**：数组用./或.\（对应元素相除），矩阵用/或\（矩阵除法）
- **乘方不同**：数组用.^（每个元素乘方），矩阵用^（矩阵乘方）
- **转置不同**：数组用.'（非共轭转置），矩阵用'（共轭转置）

#### 3. M脚本文件和M函数文件的主要区别？
1. **输入输出参数**：脚本文件不接收输入参数，也不返回输出参数；函数文件接收输入参数，也可返回输出参数
2. **变量作用域**：脚本文件的变量是全局变量，保存在基本工作空间；函数文件的默认内部变量是局部变量，工作区间不能访问
3. **第一行格式**：脚本文件没有特殊要求；函数文件第一行必须以function关键字开头
4. **适用情况**：脚本文件常用于需多次执行的一系列命令；函数文件常用于需多次执行且需要输入/输出参数的命令集合

#### 4. 截面惯性矩的定义、作用和物理意义？

**定义：** 对于构件的任意截面，取微元面积dA，该微元面积到某一轴距离的平方乘以dA，在整个截面面积上的积分。

**公式：**
- 对x轴：I_xx = ∫_A y² dA
- 对y轴：I_yy = ∫_A x² dA

**物理意义/作用：**
- 表征截面抵抗弯曲变形的能力
- 惯性矩越大，截面的抗弯刚度越大，变形越小
- 与截面的形状、尺寸有关，与材料质量无关
- 是结构力学、材料力学中计算弯曲应力和挠度的重要参数

#### 5. 平行移轴定理的内容及应用条件？

**内容：** 截面对任意轴的惯性矩，等于对其平行形心轴的惯性矩加上截面面积与两轴间距离平方的乘积。

**公式：**
- I_xx = Ī_xx + d_y² · A
- I_yy = Ī_yy + d_x² · A

**应用条件：**
1. 两个轴必须互相平行
2. 其中一个轴必须是形心轴（过截面形心）
3. d是两轴之间的垂直距离

**注意：** 过形心轴的静矩为0，即∫_A ȳ dA = 0

#### 6. for循环和while循环的区别及适用场景？

| 对比项 | for循环 | while循环 |
|--------|---------|-----------|
| 循环次数 | 已知或确定 | 未知，由条件决定 |
| 语法 | for i = 向量, 循环体, end | while 条件, 循环体, end |
| 适用场景 | 遍历数组、固定次数迭代 | 满足条件就循环，直到不满足为止 |
| 循环变量 | 自动从向量中取值 | 需在循环体内手动更新 |

### ★★ 次重点

#### 1. 符号计算与数值计算的异同？

**相同点：** 都遵循基本的数学运算规则（如加法交换律等）

**不同点：**
1. **运算对象不同**：符号计算的对象是符号（变量、函数）；数值计算的对象是具体数值（整数、浮点数）
2. **计算规则不同**：符号计算遵循数学推导和恒等变换规则；数值计算遵循数值运算和近似计算规则
3. **结果形式不同**：符号计算结果是符号表达式（解析解）；数值计算结果是具体数值（数值解）
4. **精度不同**：符号计算是精确的解析解；数值计算是近似的数值解

#### 2. MATLAB变量名命名规则及关键字？

**命名规则：**
1. 必须由一个字母开头引导
2. 后面可以跟字母、数字、下划线
3. 区分大小写（Abc ≠ ABc）
4. 有效长度为63个字符

**关键字（不能用作变量名）：**
- for、end、if、while、function、else、elseif、break、continue、return
- switch、case、otherwise、try、catch、global、persistent

#### 3. 字符串的组成及存储形式？
- **组成**：由若干个字符所组成的字符序列
- **创建方式**：将字符放入单引号中，如 'hello'
- **存储形式**：每个字符占一个元素位，按ASCII码存储
- **本质**：字符串变量本质上是一个字符数组
- **注意**：单引号创建的是字符数组，双引号创建的是字符串标量/数组

#### 4. 线性方程组Ax=b解的情况及判断方法？

设系数矩阵为A，增广矩阵为C = [A, b]，未知数个数为n：
1. **唯一解**：A为非奇异方阵，rank(A) = n。解法：X = inv(A)*B 或 X = A\b（左除法）
2. **无穷多解**：rank(A) = rank(C) < n。基础解系：null(A)，特解：pinv(A)*B
3. **无解**：rank(A) ≠ rank(C)，即矛盾方程。求最小二乘解：pinv(A)*B

#### 5. 左除法求解线性方程组唯一解的判断条件？
- 系数矩阵A必须是**方阵**
- 矩阵A必须是**非奇异**的，即满秩：rank(A) = n（n为矩阵阶数）
- 等价于：矩阵A的行列式不为零，det(A) ≠ 0
- 等价于：矩阵A可逆，存在逆矩阵inv(A)

## 四、计算题固定步骤

### ★★★ 必考点

#### 题型一：最小二乘法线性拟合

**题目形式：** 给定一组实验数据(x_i, y_i)，用最小二乘法进行线性拟合，求拟合直线方程，计算RMS误差，并绘图。

**拟合模型：** y = a₀ + a₁x

**固定步骤：**

**第1步：建立误差平方和公式**
E = Σᵢ₌₁ᴹ (yᵢ - (a₀ + a₁xᵢ))² = min
其中M是数据点个数

**第2步：对a₀和a₁分别求偏导并令其为0**
∂E/∂a₀ = 0
∂E/∂a₁ = 0

**第3步：整理得到正规方程组**
- Σyᵢ = a₀·M + a₁·Σxᵢ
- Σyᵢxᵢ = a₀·Σxᵢ + a₁·Σxᵢ²

**写成矩阵形式：**
```
[ M        Σxᵢ ] [a₀]   [ Σyᵢ   ]
[ Σxᵢ   Σxᵢ² ] [a₁] = [ Σyᵢxᵢ ]
```

**第4步：用克莱姆法则求解a₀和a₁**
- 系数行列式：D = M·Σxᵢ² - (Σxᵢ)²
- a₀ = (Σyᵢ·Σxᵢ² - Σxᵢ·Σyᵢxᵢ) / D
- a₁ = (M·Σyᵢxᵢ - Σyᵢ·Σxᵢ) / D

**第5步：写出拟合直线方程**
y = a₀ + a₁x

**第6步：计算RMS误差（均方根误差）**
RMS_error = √[ (1/M) · Σ(yᵢ - (a₀ + a₁xᵢ))² ]

**第7步：绘图**
- 先画原始数据点：用散点或圆圈标记
- 再画拟合直线：取x范围，计算对应y值，连线
- 添加图例、坐标轴标签、标题等

> ⚠️ **易错点：**
> - 二次曲线拟合有3个未知数(a₀,a₁,a₂)，需要3个方程，正规方程组是3×3的
> - 不要把Σxᵢ和Σxᵢ²搞混
> - RMS误差要开平方，不要忘了根号

**MATLAB代码模板：**
```matlab
% 给定数据
x = [x1, x2, x3, ...];
y = [y1, y2, y3, ...];
M = length(x);

% 计算各项求和
sx = sum(x);
sy = sum(y);
sxx = sum(x.^2);
sxy = sum(x.*y);

% 计算系数
deta = M * sxx - sx^2;
a0 = (sy * sxx - sx * sxy) / deta;
a1 = (M * sxy - sx * sy) / deta;

% 绘图
plot(x, y, 'o');  % 原始数据点
hold on;
xa = min(x):0.1:max(x);
ya = a0 + a1 * xa;
plot(xa, ya, '-b', 'linewidth', 2);  % 拟合直线

% 计算RMS误差
RMS = sqrt(1/M * sum((y - (a0 + a1*x)).^2));

% 标注
xlabel('x'); ylabel('y');
legend('原始数据', '拟合直线');
title('最小二乘法线性拟合');
```

#### 题型二：截面惯性矩计算（平行移轴定理）

**题目形式：** 给定组合截面（如工字钢、矩形钢），计算对形心轴的截面惯性矩，比较哪种形状更好。

**必备公式：**
- 矩形对底边：I = bh³/3
- 矩形对形心轴：Ī = bh³/12
- 平行移轴定理：I = Ī + d²·A

**固定步骤（以工字钢为例）：**

**第1步：分析组合截面的组成**
- 工字钢 = 上下两个翼缘矩形 + 中间腹板矩形
- 矩形钢 = 多个矩形叠在一起

**第2步：确定整体形心位置**
- 利用对称性：左右对称时，形心在对称轴上
- 计算y方向形心坐标：ȳ = Σ(Aᵢ·ȳᵢ) / ΣAᵢ

**第3步：对每个部分分别计算**
对每个矩形部分：
1. 计算自身对自身形心轴的惯性矩：Īᵢ = bᵢhᵢ³/12
2. 计算该部分形心到整体形心轴的距离dᵢ
3. 用平行移轴定理：Iᵢ = Īᵢ + dᵢ²·Aᵢ

**第4步：求和得到总惯性矩**
I_total = ΣIᵢ （各部分对同一轴的惯性矩相加）

**第5步：比较与结论**
- 在面积相同（材料相同）的情况下，惯性矩越大越好
- 工字钢的惯性矩通常比矩形钢大，因为材料更多分布在远离形心的位置

> 💡 **解题技巧：**
> - 先找对称轴，简化形心计算
> - 每个部分都要用平行移轴定理，不能直接加各部分对自身形心的惯性矩
> - 注意b和h：对x轴的惯性矩用h的三次方（h是垂直于x轴的方向）
> - 单位要统一

#### 题型四：矩阵运算（点乘 vs 矩阵乘）

**题目形式：** 给定两个矩阵A和B，计算A.*B和A*B的结果，并说明区别。

**固定步骤：**

**A.*B（点乘，数组运算）：**
- 条件：A和B的维度必须完全相同
- 规则：对应位置的元素相乘
- 结果：C(i,j) = A(i,j) × B(i,j)
- 结果维度：与A、B相同

**A*B（矩阵乘，线性代数）：**
- 条件：A的列数 = B的行数
- 规则：A的第i行与B的第j列对应元素相乘再求和
- 结果：C(i,j) = Σₖ A(i,k) × B(k,j)
- 结果维度：行数=A的行数，列数=B的列数

**区别总结：**
1. **运算规则不同**：点乘是对应元素相乘；矩阵乘是行乘列求和
2. **维度要求不同**：点乘要求维度相同；矩阵乘要求前一个的列数等于后一个的行数
3. **结果维度不同**：点乘结果维度与输入相同；矩阵乘结果是m×p（m行p列）
4. **数学意义不同**：点乘是逐元素运算；矩阵乘是线性变换

#### 题型六：微分方程求解（ode45）

- x₁' = x₂
- x₂' = -a·x₂ - b·x₁

end

tspan = [t_start, t_end];  % 时间区间

### 易错点提醒

#### 1. MATLAB语法易错点

> ❌
> - **加减没有点运算**：只有乘、除、乘方有点运算，加减没有.+和.-
> - **索引从1开始**：不是从0开始，A(1)是第一个元素
> - **end关键字**：表示最后一个，A(end)是最后一个元素，不要写成-1
> - **单引号 vs 双引号**：单引号是字符数组，双引号是字符串数组
> - **for循环格式**：for i = 1:5 不是 for i=1 to 5，也不需要括号
> - **矩阵维度**：点乘要求维度相同，矩阵乘要求列数=行数，不要搞混
> - **转置**：A'是共轭转置，A.'是非共轭转置，实数矩阵一样，复数不一样

#### 2. 截面惯性矩易错点

> ❌
> - **b和h搞反**：对x轴的惯性矩，h是垂直于x轴的方向，是h的三次方
> - **忘记平行移轴**：组合截面每个部分都要移到同一轴上再相加，不能直接加各自形心的
> - **1/3 vs 1/12**：对底边是1/3，对形心轴是1/12，不要记反了
> - **距离是平方**：平行移轴定理中是d的平方，不是d
> - **惯性矩与质量无关**：面积惯性矩只和形状尺寸有关，和材料质量无关

#### 3. 最小二乘法易错点

> ❌
> - **Σxᵢ 和 Σxᵢ² 搞混**：一个是x的和，一个是x平方的和，不一样
> - **RMS误差忘开根号**：均方根误差，最后要开平方
> - **二次曲线方程数**：二次拟合有3个未知数，需要3个方程，正规方程组是3×3的
> - **误差定义**：是真实值减近似值的平方和，不是绝对值和
> - **求偏导后等于0**：最小值点偏导数为0，不要忘了这个条件

#### 4. 概念辨析易错点

> ❌
> - **数组 vs 矩阵**：矩阵是特殊的二维数组，存储没区别，运算规则有区别
> - **脚本 vs 函数**：脚本没有输入输出，变量全局；函数有输入输出，变量局部
> - **符号计算 vs 数值计算**：一个是解析解，一个是数值解，不要搞混
> - **秩的判断**：唯一解是rank(A)=n，不是rank(A)=rank(C)，后者是有解的条件
> - **左除 vs 右除**：A\B是左除，解AX=B；A/B是右除，解XB=A

> - **认真审题**：根据专业选择相应题目作答（复习大纲温馨提示）
> - **计算题写步骤**：即使结果算错，步骤对了也有分
> - **编程题注意语法**：end不要漏，索引从1开始
